这今天准备学习一下自定义VIew方面的内容

在简书上看到了一篇文章讲自定义view的觉得很不错(传送门)

于是看完文章后就写了一个小demo,原文介绍挺详细，里面还有一些东西我还需要去学习

这个demo主要是写了两个方法，drawBigCircle和drawBall两个分别实现画圆环和圆球，再定义了一个执行动画的方法，其中动画用到了ValueAnimator（ValueAnimator用法）

 

 

以下为代码：

View代码

public class CIrcleView extends View{
private ValueAnimator animator;
float angle = 0;
private Paint mCirclePaint;
private Paint mBallPaint;

public CIrcleView(Context context) {
    this(context,null);
}

public CIrcleView(Context context, AttributeSet attrs) {
    this(context, attrs,0);
}

public CIrcleView(Context context, AttributeSet attrs, int defStyleAttr) {

    super(context, attrs, defStyleAttr);
    init();
}

@Override
protected void onDraw(Canvas canvas) {
    drawBigCircle(canvas);
    drawBall(canvas);

}

private void drawBigCircle(Canvas canvas){
    canvas.drawCircle(500.0f,500.0f,200,mCirclePaint);
}

private void drawBall(Canvas canvas){

    double anglethis = Math.PI /180*angle;
    float y = (float) (Math.sin(anglethis)*200);
    float x = (float) (Math.cos(anglethis)*200);
    int count = canvas.save();
    canvas.translate(500f,500f);
    canvas.drawCircle(x,y,20,mBallPaint);
    canvas.restoreToCount(count);
}

private void init() {
    mCirclePaint = new Paint();
    mCirclePaint.setAntiAlias(true);
    mCirclePaint.setStyle(Paint.Style.STROKE);
    mCirclePaint.setStrokeWidth(15.0f);
    mCirclePaint.setColor(getResources().getColor(R.color.colorAccent));

    mBallPaint = new Paint();
    mBallPaint.setAntiAlias(true);
    mBallPaint.setStyle(Paint.Style.FILL);
    mBallPaint.setColor(getResources().getColor(R.color.colorAccent));

}

@Override
protected void onAttachedToWindow() {
    super.onAttachedToWindow();
    startAnimation();
}

@Override
public void setVisibility(int visibility) {
    if (visibility == INVISIBLE || visibility == GONE){
    }else {
        clearAnimation();
        //startAnimation();
    }
    super.setVisibility(visibility);
}

private void startAnimation() {
    AccTypeEvaluator acc = new AccTypeEvaluator();
    animator = ValueAnimator.ofObject(acc,0.0f,360.0f);
    animator.setDuration(2000);
    animator.addUpdateListener(new ValueAnimator.AnimatorUpdateListener() {
        @Override
        public void onAnimationUpdate(ValueAnimator animation) {
            Float f = (Float) animation.getAnimatedValue();
            angle = f;
            invalidate();
        }
    });

    animator.setRepeatCount(ValueAnimator.INFINITE);
    animator.start();
}
}

RecyclerView 学习

RecyclerView介绍

  • RecyclerView重点是关注如何回收和复用View
  • 它不关心Item是否是显示在正确的位置，如何显示
  • 不关心Item之间是如何分隔的
  • 不关注Item增加与删除的动画效果

RecyclerView相关的重要类

  • Adapter
  • ViewHolder
  • LayoutManager（用于控制RecyclerView的显示风格）
  • ItemDecoration（控制风格效果）
  • ItemAnimator（控制Item内部增删的动画效果）

如何在项目中加入RecyclerView

  • Lru算法

    • Lru：Least Recently Used 近期最少使用算法
    • Android提供了LruCache类来实现这个算法

  • 以内存换率

      public ImageLoader(){

      int cacheSize = maxSize/4;
      //初始化Lru缓存
      mCaches = new LruCache<String,Bitmap>(cacheSize){
      @Override
protected int sizeOf(String key, Bitmap value) {
    return value.getByteCount();
}
      };
      }

    /**

    • 线程方式加载图片时用于更新UI的Handler

      private Handler handler = new Handler(){

      public void handleMessage(Message msg) {
      super.handleMessage(msg);
if(tampView.getTag().equals(mURL))
tampView.setImageBitmap((Bitmap) msg.obj);
      }
      };

    /**

    • 线程方式
      */
      public void getImage(ImageView image, final String url){
      tampView = image;
      mURL = url;
      new Thread(){
      @Override
public void run() {
    super.run();
    Bitmap bitmap = getBitmapFromURL(url);
    Message message =Message.obtain();
    message.obj = bitmap;
    handler.sendMessage(message);
}
      }.start();
      }

    /**

    • 从网络中获取图片
    • @param url
    • @return
    • @throws IOException
      */
      public Bitmap getBitmapFromURL(String url) {
      Bitmap bitmap;
      InputStream is = null;
      try {
      HttpURLConnection conn = (HttpURLConnection) new URL(url).openConnection();
is = new BufferedInputStream(conn.getInputStream());
bitmap = BitmapFactory.decodeStream(is);
conn.disconnect();
return bitmap;
      } catch (IOException e) {
      e.printStackTrace();
      }finally{
      try {
    is.close();
} catch (IOException e) {
    e.printStackTrace();
}
      }
      return null;
      }

    /**

    • 将图片添加到缓存
    • 注意：要先判读缓存中市是否已经存在这个图片
    • @param key
    • @param bitmap
      */
      private void addBitmapToCache(String key,Bitmap bitmap){
      if(mCaches.get(key) == null){
      mCaches.put(key,bitmap);
      }
      }

    /**

    • 根据key值从缓存中获取图片
    • @param key
    • @return
      */
      private Bitmap getBitmapFromCache(String key){
      return mCaches.get(key);
      }

    public void showImageByAsyncTask(ImageView imageView,String url){

    Bitmap bitmap =  getBitmapFromCache(url);
if(bitmap==null) {
    new ImageAsyncTask(imageView, url).execute(url); //缓存中没有图片，启动异步任务获取
}else {

      class ImageAsyncTask extends AsyncTask<String,Void,Bitmap>{

      private ImageView imageView;
      private String url;

      public ImageAsyncTask(ImageView imageView, String url) {
      this.imageView = imageView;
      this.url = url;
      }

      @Override
      protected Bitmap doInBackground(String… params) {
      //从网络中获取图片
      Bitmap bitmap = getBitmapFromURL(params[0]);
      String url = params[0];
      if(bitmap!=null){

      addBitmapToCache(url,bitmap); //将图片加至缓存中

      }
      return bitmap;
      }

      @Override
      protected void onPostExecute(Bitmap bitmap) {
      super.onPostExecute(bitmap);
      if(imageView.getTag().equals(url))
      imageView.setImageBitmap(bitmap);
      }
      }

    }

学习Android异步任务-AsyncTask

异步任务-AsyncTask

  • 构建Async子类的参数
    AsyncTask<Params,Progress,Result>是一个抽象类
    通常用于被继承，继承AsyncTask需要指定以下三个泛型参数:
  1. Params: 启动任务时输入参数的类型。
  2. Progress:后台任务执行中返回进度值的类型
  3. Result: 后台执行任务完成后返回结果类型
    -

  • 构建AsyncTask子类的回调方法

    • doinBackground: 必须重写，移步执行后台线程将要完成的任务
    • doPreExecute: 执行后台耗时操作前被调用，通常用户完成一些初始化操作
    • onPostExecute: 当doinBackground()完成后，系统会调用onPostExecute()方法，并将doinBackground()方法返回的值传给该方法
    • onProgressUpdate: doInBackground()方法中调用publicProgress()方法，更新任务的执行进度后，就会触发该方法

<pre>`public class LoadImage extends AppCompatActivity {

private ImageView imageView;
private ProgressBar progressBar;
private static final  String IMAGE_URL =
        "http://cdnq.duitang.com/uploads/item/201412/17/20141217131537_v3ExX.jpeg";

@Override
protected void onCreate(Bundle savedInstanceState) {
    super.onCreate(savedInstanceState);
    setContentView(R.layout.activity_load_image);
    initView();
    ImageTask task = new ImageTask();
    task.execute(IMAGE_URL);
}

private void initView() {
    imageView = (ImageView) findViewById(R.id.image1);
    progressBar = (ProgressBar) findViewById(R.id.progress);
    progressBar.setMax(50);
}

class ImageTask extends AsyncTask&lt;String,Integer,Bitmap&gt;{

    @Override
    protected void onPreExecute() {
        super.onPreExecute();
        progressBar.setVisibility(View.VISIBLE);
    }

    @Override
    protected Bitmap doInBackground(String... params) {
        Bitmap bitmap ;
        String url = params[0];
        URLConnection conn ;
        InputStream is;
        try {
            conn =  new URL(url).openConnection();

            is = conn.getInputStream();

            BufferedInputStream buffer = new BufferedInputStream(is);
            for(int i = 0;i&lt;50;i++){
                Thread.sleep(50);
                publishProgress(i);
            }

            bitmap = BitmapFactory.decodeStream(buffer);
            return bitmap;
        } catch (IOException e) {
            e.printStackTrace();
        } catch (InterruptedException e) {
            e.printStackTrace();
        }

        return null;
    }

    @Override
    protected void onPostExecute(Bitmap bitmap) {
        super.onPostExecute(bitmap);
        progressBar.setVisibility(View.INVISIBLE);
        imageView.setImageBitmap(bitmap);

    }

    @Override
    protected void onProgressUpdate(Integer... values) {
        super.onProgressUpdate(values);
        progressBar.setProgress(values[0]);
    }
}
}

以下注意事项来自：慕课网

使用AsyncTask的注意事项

① 必须在UI线程中创建AsyncTask的实例.

② 只能在UI线程中调用AsyncTask的execute方法.

③ AsyncTask被重写的四个方法是系统自动调用的,不应手动调用.

④ 每个AsyncTask只能被执行(execute方法)一次,多次执行将会引发异常.

⑤ AsyncTask的四个方法,只有doInBackground方法是运行在其他线程中,其他三个方法都运行在UI线程中,也就说其他三个方法都可以进行UI的更新操作.

SharedPreferences工具类for Android

 

import android.content.Context;
import android.content.SharedPreferences;

import java.util.Map;

/**

  • Created by addi on 2016/2/29 0029.
  • sharepreferences工具类
    /
    public class SPUtils {
    /*
  • 保存在手机里面的sp文件名
    */
    public static final String FILE_NAME = “my_sp”;

/**

  • 保存数据
    */
    public static void put(Context context,String key, Object obj){
    SharedPreferences sp = context.getSharedPreferences(FILE_NAME,context.MODE_PRIVATE);
    SharedPreferences.Editor editor = sp.edit();
    if(obj instanceof Boolean){
    editor.putBoolean(key,(Boolean) obj);
    }
    if(obj instanceof Float){
    editor.putFloat(key,(Float)obj);
    }
    if (obj instanceof Integer){
    editor.putInt(key,(Integer)obj);
    }
    if(obj instanceof Long){
    editor.putLong(key,(Long)obj);
    }
    if(obj instanceof String){
    editor.putString(key,(String)obj);
    }
    editor.commit();
    }

/**

  • 获取指定数据
    */
    public static Object get(Context context,String key,Object defaultObj){
    SharedPreferences sp = context.getSharedPreferences(FILE_NAME,context.MODE_PRIVATE);
    if(defaultObj instanceof Boolean){
    return sp.getBoolean(key,(Boolean)defaultObj);
    }
    if(defaultObj instanceof Float){
    return sp.getFloat(key,(Float)defaultObj);
    }
    if(defaultObj instanceof Integer){
    return sp.getInt(key,(Integer)defaultObj);
    }
    if(defaultObj instanceof Long){
    return sp.getLong(key,(Long)defaultObj);
    }
    if(defaultObj instanceof String){
    return sp.getString(key,(String)defaultObj);
    }
    return null;
    }

/**

  • 删除指定数据
    */
    public static void remove(Context context,String key){
    SharedPreferences sp = context.getSharedPreferences(FILE_NAME,context.MODE_PRIVATE);
    SharedPreferences.Editor editor = sp.edit();
    editor.remove(key);
    editor.commit();
    }

/**

  • 返回所有键值对
    */
    public Map getAllKey(Context context){
    SharedPreferences sp = context.getSharedPreferences(FILE_NAME,context.MODE_PRIVATE);
    Map map = sp.getAll();
    return map;
    }

/**

  • 删除所有数据
    */
    public static void clear(Context context){
    SharedPreferences sp = context.getSharedPreferences(FILE_NAME,context.MODE_PRIVATE);
    SharedPreferences.Editor editor = sp.edit();
    editor.clear();
    editor.commit();
    }

/**

  • 检查对应的数据是否存在
    */
    public static boolean contains(Context context,String key){
    SharedPreferences sp = context.getSharedPreferences(FILE_NAME,context.MODE_PRIVATE);
    return sp.contains(key);
    }

}
