Android 5.0 Button 按钮水纹效果的适配问题-解决自定义button颜色时水波纹的显示

从5.0开始,一些控件的点击时候默认是有水纹效果的如下图

以Button举例可以按照如下使用:
注意是android:theme

<span class="hljs-tag">&lt;<span class="hljs-title">Button</span>  
    <span class="hljs-attribute">android:layout_width</span>=<span class="hljs-value">"wrap_content"</span>
    <span class="hljs-attribute">android:layout_height</span>=<span class="hljs-value">"wrap_content"</span>
    <span class="hljs-attribute">android:text</span>=<span class="hljs-value">"Button"</span>
    <span class="hljs-attribute">android:theme</span>=<span class="hljs-value">"@style/MyButton"</span>/&gt;</span>`</pre>

自定义样式:

&lt;style <span class="hljs-property">name</span>=<span class="hljs-string">"MyButton"</span> parent=<span class="hljs-string">"Theme.AppCompat.Light"</span>&gt;  
        &lt;<span class="hljs-property">item</span> <span class="hljs-property">name</span>=<span class="hljs-string">"colorControlHighlight"</span>&gt;@color/indigo&lt;/<span class="hljs-property">item</span>&gt;
        &lt;<span class="hljs-property">item</span> <span class="hljs-property">name</span>=<span class="hljs-string">"colorButtonNormal"</span>&gt;@color/pink&lt;/<span class="hljs-property">item</span>&gt;
    &lt;/style&gt;

但是这样在5.0以下的系统,这样就没有效果,按钮会变成默认的灰色状态,点击按钮也不会有颜色的状态区别
为了兼容5.0以下的设备,我们可以利用seletor(勘误:button用theme在5.0以下也是有按压变色的,虽然没有水波纹,下面的方法仍然适用其他控件的适配)

                    android:id=<span class="hljs-string">"<span class="hljs-variable">@+</span>id/confirm"</span>
                    style=<span class="hljs-string">"<span class="hljs-variable">@style</span>/GreenButton"</span>
                    android:text=<span class="hljs-string">"<span class="hljs-variable">@string</span>/confirm"</span> /&gt;

<pre class="prettyprint">`  &lt;style <span class="hljs-property">name</span>=<span class="hljs-string">"GreenButton"</span>&gt;
        &lt;<span class="hljs-property">item</span> <span class="hljs-property">name</span>=<span class="hljs-string">"android:layout_width"</span>&gt;match_parent&lt;/<span class="hljs-property">item</span>&gt;

        &lt;<span class="hljs-property">item</span> <span class="hljs-property">name</span>=<span class="hljs-string">"android:background"</span>&gt;@drawable/selector_green_button&lt;/<span class="hljs-property">item</span>&gt;

我们可以在drawable和drawable-v21下新建同一份文件selector_green_button.xml

`<?xml version=”1.0” encoding=”utf-8”?>
    <selector xmlns:android=“http://schemas.android.com/apk/res/android“>

    <span class="hljs-tag">&lt;<span class="hljs-title">item</span> <span class="hljs-attribute">android:drawable</span>=<span class="hljs-value">"@color/deep_green"</span> <span class="hljs-attribute">android:state_pressed</span>=<span class="hljs-value">"true"</span> /&gt;</span>

有个问题需要注意,如果的想要你的波纹没有按压的时候为透明状态,下面的写法是无效的.会导致按压没有效果

<pre class="prettyprint">`<span class="hljs-comment">&lt;!--v21下 --&gt;</span>
<span class="hljs-pi">&lt;?xml version="1.0" encoding="utf-8"?&gt;</span>
<span class="hljs-tag">&lt;<span class="hljs-title">ripple</span> <span class="hljs-attribute">xmlns:android</span>=<span class="hljs-value">"http://schemas.android.com/apk/res/android"</span>
    <span class="hljs-attribute">android:color</span>=<span class="hljs-value">"@color/deep_green"</span>&gt;</span>//这里是扩散水波纹的色值
    <span class="hljs-tag">&lt;<span class="hljs-title">item</span> <span class="hljs-attribute">android:drawable</span>=<span class="hljs-value">"@android:color/transparent"</span> /&gt;</span>//这里你用透明色和透明的图片都是没有效果的
<span class="hljs-tag">&lt;/<span class="hljs-title">ripple</span>&gt;</span>`</pre>

正取的做法如下:

<span class="hljs-comment">&lt;!--v21下 --&gt;</span>
    <span class="hljs-pi">&lt;?xml version="1.0" encoding="utf-8"?&gt;</span>
    <span class="hljs-tag">&lt;<span class="hljs-title">ripple</span> <span class="hljs-attribute">xmlns:android</span>=<span class="hljs-value">"http://schemas.android.com/apk/res/android"</span>
        <span class="hljs-attribute">android:color</span>=<span class="hljs-value">"@color/deep_green"</span>&gt;</span>//这里是扩散水波纹的色值
         <span class="hljs-tag">&lt;<span class="hljs-title">item</span> <span class="hljs-attribute">android:id</span>=<span class="hljs-value">"@android:id/mask"</span> <span class="hljs-attribute">android:drawable</span>=<span class="hljs-value">"@color/white"</span> /&gt;</span>里面的色值可以任选一个
    <span class="hljs-tag">&lt;/<span class="hljs-title">ripple</span>&gt;</span>

android:id=”@android:id/mask”会让系统并不会真的绘制,并告知波纹的绘制边界

如果写成下面,波纹的绘制范围会超出控件的边界
<pre class="prettyprint">`<span class="hljs-comment">&lt;!--v21下 --&gt;</span>
<span class="hljs-pi">&lt;?xml version="1.0" encoding="utf-8"?&gt;</span>
<span class="hljs-tag">&lt;<span class="hljs-title">ripple</span> <span class="hljs-attribute">xmlns:android</span>=<span class="hljs-value">"http://schemas.android.com/apk/res/android"</span>
    <span class="hljs-attribute">android:color</span>=<span class="hljs-value">"@color/deep_green"</span>&gt;</span>//这里是扩散水波纹的色值
<span class="hljs-tag">&lt;/<span class="hljs-title">ripple</span>&gt;</span>

如上,这样设置后5.0以下的设备具有按钮按下变色的效果,5.0以上就具有水波纹效果.其他控件同理
